# my-brand-film-director

中文注释：自制 Skill｜品牌广告概念导演

## 安装位置

Windows PowerShell 用户级路径：

```powershell
mkdir $HOME\.agents\skills\my-brand-film-director
copy .\SKILL.md $HOME\.agents\skills\my-brand-film-director\SKILL.md
```

Mac / Linux 用户级路径：

```bash
mkdir -p ~/.agents/skills/my-brand-film-director
cp SKILL.md ~/.agents/skills/my-brand-film-director/SKILL.md
```

## 启动方式

在 Codex 里输入：

```text
/skills
```

然后选择：

```text
my-brand-film-director
```

也可以直接输入：

```text
$my-brand-film-director
```

客户是做家居服面料的，价格中等，面向家庭消费者，想要室外阳光、简单高级、舒服的风格。主要做竖屏TVC，9:16，8秒和15秒两版，Logo结尾出现。
